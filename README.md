### Using VS Code

1. **Open the workspace**: Double-click `art-109.code-workspace` to open the project in VS Code
2. **Live Server**: 
   - Install the "Live Server" extension if you haven't already
   - Right-click any HTML file → "Open with Live Server"
   - Or click the "Go Live" button in the bottom right corner

### Viewing Your Site

- **Local**: Use Live Server (usually at `http://localhost:5500`)
- **GitHub Pages**: Your site is published at `https://rosalva003.github.io/art-109/`

### Tips

- Files auto-save after 1 second
- HTML files format automatically on save
- Use the workspace file for the best VS Code experience
